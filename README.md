# PyWar

## GitHub Commit Syntax

    Bracket -> Colon -> Hyphen -> Change

### Example Commit

    [Fix] Documentation: Player Class - def __init__

|   Syntax    | Options                                   |
| ----------- | -----------                               |
| Bracket     | Fix                                       |
| Colon       | Documentation, Testing, DocTest, Creation |
| Hyphen      | Classes, README.md                           |
| Change      | Functions                                 |
