# PyWar

## GitHub Commit Syntax

    Bracket -> Colon -> Hyphen -> Change

### Example Commit

    [Fix] Documentation: Player Class - def __init__

|   Syntax    | Options                                   |
| ----------- | -----------                               |
| Bracket     | Creation, Fix, Update                     |
| Colon       | DocTest, Documentation, Testing           |
| Hyphen      | Classes, README.md, Sections              |
| Change      | Functions                                 |
